# Badge

## Usage

```js
import { Badge } from 'asc-web-components';

<Badge />;
```

#### Description

Item notify badge.

| `number`          | `number` |          |        | 0         | Number value          |

| `color`           | `string` |          |        | '#FFFFFF' | HTML font color       |
| `fontSize`        | `string` |          |        | '11px'    | HTML font size        |
| `fontWeight`      | `number` |          |        | '800'     | HTML font weight      |
| `onClick`         | `func`   |          |        |           | onClick event         |
