# ToggleContent

ToggleContent allow you to adding information, which you may hide/show by clicking header

### Usage

```js
import { ToggleContent } from "asc-web-components";
```

```jsx
<ToggleContent>
  <span>Some text</span>
</ToggleContent>
```
