# RowContainer

## Usage

```js
import { RowContainer } from 'asc-web-components';
```

#### Description

Container for rows

#### Usage

```js
<RowContainer manualHeight='500px'>
    {children}
</RowContainer>
```

#### Properties

| Props            | Type       | Required | Values | Default | Description                                               |
| ---------------- | ---------- | :------: | ------ | ------- | --------------------------------------------------------- |
| `manualHeight`        | `string`     | -        |        | -     |                                                    |
| `itemHeight`        | `number`     | -        |        | 50     |                                                    |
